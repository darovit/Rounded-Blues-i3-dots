## Various Info Things


* Wallpaper: #008080
* OS: Arch
* WM: i3-gaps rounded
* Compositor: compton
* Bar: Polybar
* Shell: ZSH 
* Terminal: Kitty
* Terminal font: Source Code Pro
* Resource monitor: gtop
* Audio Player: NCMPCPP with mopidy-spotify
* Visualizer: cava
* Discord client: 6cord
* Other: roku-cli to control roku
